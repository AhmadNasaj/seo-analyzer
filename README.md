# SEO Analyzer

A simple SEO Analyzer tool to check your website SEO issues like missing meta tags, headings, and image alt text.

## Features
- Check page title & meta description
- Analyze heading structure (H1, H2, H3)
- Check image alt attributes
- Suggest improvements for better SEO

## How to Use
1. Open the tool in your browser: [SEO Analyzer](https://nasajtools.com/tools/seo/seo-analyzer.html)
2. Paste the URL of the page you want to analyze.
3. Click "Analyze" and review the suggestions.
4. Make improvements and check again.

## Why I Created This
I wanted a simple, practical tool for developers and content creators to quickly improve website SEO without overthinking it.
